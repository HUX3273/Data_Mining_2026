# Data_Mining_2026

graduate course project

## 基于多源异构数据整合的“双一流”高校高层次人才梯队分析

### 项目简介

本项目为**数据挖掘课程终期项目**，聚焦 2025 年度 **杰青（国家杰出青年科学基金获得者）** 与 **优青（国家优秀青年科学基金获得者）** 的多源异构数据整合与高校梯队分析。

核心流程：
1. **数据采集** — 从 Unionpub 学术、科学家之家、浙江微流纳米等 3 个来源采集原始名单；
2. **数据审计** — 识别跨源冲突、重复姓名、口径差异；
3. **可信度融合** — 按来源优先级投票，生成可追溯的融合数据集；
4. **聚类分析** — 基于 K-Means 对高校进行分型（杰青主导型 / 领军人才密集型）；
5. **可视化** — 散点图、热力图、分布图、饼图。

### 关键发现

1. **头部效应显著**：中国科学院（37 杰青 + 20 优青）、南京大学（13 杰青）、北京大学（10 杰青）等构成绝对第一梯队；
2. **杰青 > 优青**：2025 年杰青总数（287）显著高于优青（199），反映杰青项目竞争更激烈；
3. **两极分化**：仅约 6% 的高校形成“领军人才密集型”格局，绝大多数高校人才储备有限；
4. **数据审计价值**：通过显式记录冲突、来源和优先级，避免了简单合并导致的系统性偏差。
